# Выгрузка ФККО и сопутствующий парсер

Данные актуальны на 25 марта 2026 г.

## Поля файла `fkko.csv`
`code` — Код отхода. Последняя цифра соответствует классу опасности, ноль — для категорий.  
`name` — Наименование отхода  
`link` — постфикс для ссылки на сайт Росприроднадзора. Чтобы получить валидную ссылку, добавьте к началу `https://rpn.gov.ru/fkko`

## Парсер
Парсер `fkko.py` пробегается по всем страницам сайта `https://rpn.gov.ru/fkko/` и выгружает список отходов с кодами и наименованиями. Актуально до тех пор, пока не изменится API сайта. В случае изменений — Feel free to commit!  
Требования: `polars 1.39.3` `bs4 0.0.2`
